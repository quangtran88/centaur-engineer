# Learning Journal

Append-only. Newest entry last. Format: `## YYYY-MM-DD` + 1–3 bullets.

Rules of the journal:

- Write it the day it happens. A journal reconstructed from memory is a story, not a record.
- 1–3 bullets per entry. If you need more, you are writing a doc — link it instead.
- Record what you *learned or decided*, not what you did. "Read DDIA ch. 6" is a status; "replication lag explains our stale-read bug from {{a real incident of yours}}" is an entry.
- The weekly review reads this file and grades 2–3 real drill entries crisp/fuzzy/blank. Give it material.
